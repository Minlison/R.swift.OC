# R.swift.OC
R.swift 支持 Objective-C 语言, 只针对个人项目进行修改, 请访问源码地址 [R.swift](https://github.com/mac-cain13/R.swift)

当前目录下的 rswift 为编译好的文件

进入目录 [RSwiftOC](./RSwiftOC) 下 在终端执行 `swift build` 命令
